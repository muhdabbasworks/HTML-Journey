## Key Concepts.
### Table Tag in HTML.
- `<Table>` Tag in HTML is used to represent real life list data.
- `<tr>` tag is used to display the row of table.
- `<th>` tag is used to display the header data of the table.
- `<td>` tag is used to display the data of the table.

#### Caption Tag.
- `<caption>` tag is used to add caption to the Table

#### Thead tag and Tbody Tag.
- `<thead>` tag is used to specify the Header of the Table.
- `<tbody>` tag is used to specify the Body of Table

#### Colspan Attribute.
- Colspan attribute is used to define the numbers of columns a single table cell should span horizontally.

### Form in HTML.
- `<form>` tag in HTML is used to collect data from the user and send that data to the server.

#### Action Attribute in Form Tag.
- Action attribute in form tag is used where the form data will be send after the submition of form.

#### Form Element.
- Form element is used to allow the user to enter, select, or submit data inside a form.

#### Text Area Tag.
- `<textarea>` tag in form tag is used to allow the user to type multi-line text such as feedback, comment, or messages.

#### Common form elements include:
- Text field
- Password field
- Radio button
- Checkbox
- Textarea
- Button

### Label Tag.
- `<label>` in form tag is used to describe or name a form input so user understand what the input is for.

#### For Attribute.
- For attribute connect the Label tag to a specific input element using the `id`

#### Why it is important:
- Clicking the label selects the input
- Improves accessibility
- Makes forms easier to use
