# The First days Concepts in HTML:

About HTML:
HTML stands for Hyper Text Markup Language.
HTML is a Common-Case Insansetive Language.
Used to create and structure content on the Website.

First HTML File.
index.htlm is the default name for a website homepage (1st page).

Basic HTML page:
<!DOCTYPE html> (tells browser you are using HTML 5)
<html lang="en"> (about language)
<head> (root of an document)
    <title>Title</title> (Page Title)
</head>
<body> (Contains all the data rendered by the browser)
    <p>Hello world!</p> (Paragraph tag)
</body>
</html>

Quick Points:
HTML tag is a parent of head and body tag
Most of html elements have opening & closing tags
with content in between
Some tags have no content in between, eg - <br>
We can use inspect element/view page source to edit html

HTML Tags:
The component used to design the structure of the websites are called HTML Tags. Today i learnt the below tags:

Heading Tag.
used to display deading in HTML.These are h1-h6. Their importance are checked through ranking. h1 (most important), h2 (less important than h1 more important than h3) and so on.

Paragraph Tag:
Used to add paragraphs in HTML
<p> This is a sample paragraph </p>

Anchor Tag
Used to add links to your page
<a href="https://google.com"> Google </a>

Image Tag
Used to add images to your page
<img src="/image.png" alt="Random Image">
